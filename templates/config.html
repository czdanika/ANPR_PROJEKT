MQTT Kapcsolat
Ha be van kapcsolva, minden felismeréskor JSON üzenetet küld a megadott broker felé.
A Home Assistant gépe vagy önálló MQTT broker (pl. Mosquitto) IP-címe.
Alapértelmezett: 1883. TLS esetén általában 8883.
Topik beállítások
Az MQTT topik, amire minden felismerés adata érkezik JSON formában.
Példa: { "license_plate": "ABC-123", "known": true, "friendly_name": "Saját autó", "image_url": "http://..." }
Ha be van kapcsolva, a broker megtartja az utolsó üzenetet. HA így mindig látja az utolsó rendszámot.
Ha be van kapcsolva, rendszámonként külön topikra is küld üzenetet.
Pl. anpr/plates/ABC-123
Home Assistant integráció
Ha be van kapcsolva, az ANPR szenzor automatikusan megjelenik HA-ban.
Beállítások → Eszközök és szolgáltatások → MQTT alatt jelenik meg az ANPR Rendszer eszköz.
Alapértelmezett: homeassistant.
A Pi elérési címe a hálózaton. Az MQTT payload-ban a kép mezőnél teljes URL-t küld.
Automatikus frissítés
Ha be van kapcsolva, az eseménylista automatikusan újratöltődik a megadott időközönként.
Minimum 5 másodperc. Ajánlott: 30.
Mennyi eseményt töltjön be egyszerre. Ajánlott: 50–200.
Kép tárolás
A kamera által készített képek automatikus törlésének szabályai. A 0 érték azt jelenti, hogy az adott limit nincs alkalmazva.
Ennyi napnál régebbi képeket törli. 0 = nincs kor alapú törlés.
Ha ennél több kép van, a legrégebbieket törli. 0 = nincs limit.
Webhook – Ismert rendszámok
Ismert rendszám észlelésekor HTTP POST kérést küld a megadott URL-ekre JSON payload-dal.
HA webhook URL: http://<HA_IP>:8123/api/webhook/<webhook_id>
Ha be van kapcsolva, POST kérést küld minden releváns észlelésnél.
Ismert rendszám + érkezés irány esetén hívja meg.
E-mail értesítés
Rendszám észleléskor e-mailt küld a megadott SMTP szerveren keresztül. Gmail esetén: host smtp.gmail.com, port 587, és engedélyezd az App jelszót.
Ha be van kapcsolva, minden egyező eseménynél e-mailt küld.
Üresen hagyva a felhasználónevet használja.
Több cím vesszővel elválasztva.
A kamerakép csatolmányként is megérkezik.
Kapu / Relé triggerelés
Ismert rendszám észlelésekor közvetlenül meghív egy HTTP végpontot – pl. Shelly relét.
Shelly Gen1: http://<shelly_ip>/relay/0?turn=on → módszer: GET
Shelly Gen2+: http://<shelly_ip>/rpc/Switch.Set → módszer: POST, törzs: {"id":0,"on":true}
Ismert rendszám észlelésekor meghívja a megadott relé URL-t.
Shelly Gen2+: {"id":0,"on":true}  |  Auto-off 2 mp: {"id":0,"on":true,"toggle_after":2}
Home Assistant konfiguráció
Másold be az alábbi YAML-t a Home Assistant konfigurációjába. A generált kód mindig a jelenlegi beállításaidat tükrözi.
Kamera snapshot URL (utolsó kép, mindig friss)
Per-kamera szűrés: URL + ?ip=192.168.0.x
Statikus kép URL – latest.jpg (HA Generic Camera)
Minden beérkező kameraképnél automatikusan frissül. Jelszó nélkül elérhető.
HA YAML: still_image_url: <fenti URL>
